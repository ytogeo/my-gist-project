# 武汉市交通健康信息可视化平台使用说明

（大三GIST课程作业，个人学习向）
本系统尝试集成Cesium等多种第三方库，实现研究区武汉市交通健康状况的可视化。该方法以网络爬虫爬取的武汉市高德交通健康大数据作为源数据，使用Cesium构建三维场景，并引入Echart、高德地图JS API等，实现交通地理数据在多个维度上的可视化；本方法搭建的系统以三维场景作为主视角，辅以多种数据可视化方式，互相补足，从而更加直观、全面地向用户展示武汉市的交通健康状况。

## 实现功能
1. 后端数据爬取：使用浏览器访问高德大数据平台（武汉市页面），通过浏览器开发者工具进行抓包，获取该网页上动态数据的来源URL。之后，在NodeJS环境下的后端代码中基于Cherrio和iconv-lite模块实现爬虫，基于Request模块和Express框架实现网络请求发送与服务器搭建。
2. 数据图表渲染：使用Echart对后端传来的数据以图表方式进行可视化.
3. 二维地图展示：使用高德地图JS API显示二维地图/实时路况/路径规划结果。
4. 三维场景构建：使用Cesium对后端数据进行可视化/将高德API传来的路径规划结果（JSON）绘制在三维场景中

## 使用方法

在VScode的终端中或是在cmd中将路径切换到文件夹根目录，执行命令：`node server.js` 以启动服务器。<br>
在浏览器中输入 <http://localhost:5505/index.html> 访问本系统首页。

## 关闭服务器
在VScode的终端中或是在cmd中将路径切换到文件夹根目录，输入：
> Crtl+C

以关闭服务器。
